# 整数反转
## 第一遍的思路
	完全没有自己的思路。因为还需要判断反转后的整数是否溢出。

## 官方题解思路
    我们可以一次构建反转整数的一位数字。在这样的时候，我们可以预先检查原整数附加另一位是否会导致溢出。
    反转整数的方法可以与反转字符串进行类比。
    我嫩想重复"弹出"x的最后以为数字，并将他"推入"到rev的后面，最后，rev将与x相反。
    要在没有辅助堆栈/数组的帮助下"弹出"和"推入"数字，我们可以使用数学方法。
    //pop operation
    pop = x % 10;
    x /= 10;

    //push operation
    temp = rev * 10 + pop;
    rev = temp;
